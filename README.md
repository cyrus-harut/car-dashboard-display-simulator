# Car dashboard display simulator

## Описание
Симулятор дисплея приборной панели автомобиля - это клиентское приложение для браузера, имеющее два направления применения: 
 1. С одной стороны помогает начинающим QA-инженерам оттачивать навыки написания тест-кейсов,
 2. С другой стороны, его можно использовать для оценки практических навыков QA-инженеров.

Приложение не имеет серверной части и развёрнуто на бесплатном хостинге в демонстрационно-эксплуатационных целях по адресу <http://k99969ca.beget.tech/>
