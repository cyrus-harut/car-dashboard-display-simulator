# Car dashboard display simulator

## Описание
Симулятор дисплея приборной панели автомобиля - это приложение для браузера, имеющее два направления применения: 
 1. С одной стороны помогает начинающим QA-инженерам оттачивать навыки написания тест-кейсов,
 2. С другой стороны, его можно использовать для оценки практических навыков QA-инженеров.

Приложение не имеет серверной части и развёрнуто на бесплатном хостинге в демонстрационно-эксплуатационных целях по адресу <http://k99969ca.beget.tech/>

Страница приложения содержит ссылку на требования, по которой скачивается PDF документ со спецификацией, описывающей требования к поведению элементов абстрактного дисплея приборной панели автомобиля. Также здесь присутствует блок, имитирующий поведение самого дисплея и блок управления, через который можно задавать соответствующие входные параметры для формирования ожидаемого результата в соответствии с описанными требованиями.

## Применение

- Для начинающего QA-инженера может быть некоторой проблемой найти ресурс для оттачивания навыков применения техник тест-дизайна. С одной стороны он может зайти на любой ресурс в интернете и начать его "проверять". Однако, этот процесс сведётся только к так называемому исследовательскому тестированию, т.к. доступа к реальным требованиям к приложению, на которое он пытается написать проверки, у него не будет. Здесь же есть чётко описанная документация с требованиями, каждое из которых имеет свой идентификатор, что позволяет попрактиковаться не просто в написании тесткейсов, но также и в работе с трассировкой. Всё, что нужно, - птосто скачать требования, ознакомиться с ними, написать свою тестовую процедуру и здесь же её прогнать.

- Применение со стороны опытных QA находится в возможности формирования различных заданий по проверке компонентов дисплея для оценки уровня компетенций других сотрудников. Применение возможно, как при рекрутинге, так и при оценке грейда сотрудников.
